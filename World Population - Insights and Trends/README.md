- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Google Colab

The analysis includes:

<ins> Initial Data Analysis (IDA) </ins>

<ins> Exploratory Data Analysis (EDA) </ins>

- Correlation
- Visualisations of patterns and behaviour
