# A-Modern-Analysis-of-Patterns-Behaviour-Montgomery-County-Crime
## Overview

This project presents a comprehensive Initial Data Analysis (IDA) and Exploratory Data Analysis (EDA) of crime incidents reported in Montgomery County, USA, from 2018 to 2022.
The goal is to uncover crime patterns, behavioural trends, hotspots, and key insights that support improved public safety strategies and preventive measures.

## Dataset Description

*Access dataset here: https://catalog.data.gov/dataset/crime*

This study is based on a crime incidents database created and maintained by the Montgomery County Police Department, fully compliant with the National Incident-Based Reporting System (NIBRS).

Includes every reported crime incident from 2018 to 2022

Protects anonymity by removing personally identifiable information

Provided in structured tabular form (CSV or similar)

Contains rich details such as:

- Victim information
  
- Date & time
  
- Location
  
- Crime category/offense type

The dataset’s structured nature allows for efficient cleaning, manipulation, and visualisation using modern data analysis tools.

## Project Workflow

The analysis includes:

<ins> Initial Data Analysis (IDA) </ins>

- Data quality checks

- Missing value inspection

- Outlier detection

- Understanding variables & structure

<ins> Exploratory Data Analysis (EDA) </ins>

- Univariate, bivariate & multivariate analysis

- Crime distributions across time, locations, and categories

- Identification of hotspots

- Visualisations of patterns and behaviour
